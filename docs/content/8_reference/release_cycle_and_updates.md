# Updates and Release Cycle

We aim to publish a new release every 2 weeks.  
Each release will include important information to help you understand what has changed and how to update your installation.

---

## Release contents
Every release will include:

- General release notes
- Changes and new features
- Breaking changes (if any)
- Upgrade instructions (specific to that release)

---

## Upgrade information
For now, only general upgrade guidelines will be available once more releases are published.  
Until then, please refer to the individual release notes for any update instructions.

---

## Why a regular release cycle?

- Predictable updates for users and maintainers
- Faster delivery of new features and improvements
- Easier tracking of breaking changes and migrations
